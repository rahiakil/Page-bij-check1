## Deployment to GitHub Pages

### Method 1: Automatic Deployment (Recommended)

1. Push this repository to GitHub
2. Go to your repository settings
3. Navigate to "Pages" in the left sidebar
4. Under "Source", select your branch (usually `main` or `master`)
5. Select the root folder (`/`)
6. Click "Save"
7. Your site will be available at `https://[username].github.io/[repository-name]`

### Method 2: Manual Deployment

1. Ensure `index.html` is in the root directory
2. Push to GitHub
3. Enable GitHub Pages in repository settings
4. Your site will be live within a few minutes

## Customization

### Colors
The design uses CSS custom properties (variables) defined in the `:root` selector. Modify these to change the color scheme:

```css
--primary-blue: #0066CC;
--dark-blue: #004499;
--light-blue: #E6F2FF;
```

### Content
All content is in `index.html`. Simply edit the HTML to update:
- Company information
- Funding amounts
- Valuation projections
- Market data
- Roadmap milestones
